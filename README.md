
# Hangman Game with Flutter

This game developed with Flutter


## Features

- Retro font letters 
- Current Points
- Player Lives
- Cross platform
- Splash Screen with game Logo
- Basic Game Logic, You won, You Lost with Points
- Game sounds and Mute option
